Welcome to Deforestation prediction

Helping to protect our planet.

Analysis : -

Select two images of the same location taken at different time intervals. Click the "Detect Deforestation" button to view the images with the detected deforestation rate.


Prediction : -

Select a state from the "subnational1" dropdown list and a threshold value from another dropdown. Click the "Prediction" button to see the deforestation prediction rate in a graph format.


Map : -

Click the "Map" button to view the map of India with parks and forests.
